# OfficeVerse User Manual

Welcome to **OfficeVerse**, a virtual office environment designed for remote collaboration and productivity.

## Getting Started

1.  Open the application in your web browser (default: `http://localhost:5173`).
2.  Enter your details in the login screen (Name, Avatar, Room Code).
3.  Click "Enter Office" to join the virtual workspace.

## Controls

| Action | Key(s) / Input | Description |
| :--- | :--- | :--- |
| **Move** | `Arrow Keys` | Move your character (Up, Down, Left, Right). |
| **Interact (Zone)** | `F` | Interact with office zones (e.g., Sit at desk, Open Meeting). |
| **Interact (Player)** | `E` | Interact with nearby players (e.g., Private Chat). |
| **Help** | `?` (Shift + `/`) | Toggle the shortcuts help overlay. |
| **Close Menu** | `ESC` | Close any open menus or popups. |
